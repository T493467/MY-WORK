What is class And id Attributes and its purpose.

> class Attribute:

To apply editing on different lines, objects and texts by using class Attribute.

Example syntax

< head >
< style >
.class assigned name or letter {
Enter editing type that you want.
}
< /style >
< /head >


> id Attribute:

To apply editing on different lines, objects and texts by using id Attribute.

Example syntax:

< head >
< style >
#id assigned name or letter {
Enter editing type that you want.
}
< /style >
< /head >
